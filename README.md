# GRUBHUB Prototype

## Distributed service-oriented food ordering application using MERN stack and Kafka MQ

### Roles

1. Customer
** Search
** Cart
** Place orders
** Track orders
** Chat with restaurant owner

2. Restaurant owner
** Manage menu - Add/Remove sections, Add/remove Items, Edit items(price, picture etc.)
** Track orders
** Update order status
** Chat with customer
